# LeetCode Optimal Solutions

Add original local implementations here using the same category and file stem as
`docs/algorithms/leetcode`.

Example:

```text
docs/algorithms/leetcode/array/1_two-sum.md
optimal_solutions/leetcode/array/1_two-sum.py
```

Use cOde(n)'s project-native interface:

```python
def solve(...):
    ...
```

Do not use LeetCode's `class Solution` wrapper in these files. For design
problems, keep `solve(...)` as the public entry point and use private helpers
only when they make the implementation clearer.
