## Problem Statement

Complete the code for Binary Search to find the index of an element $K$ in the given array $A$.
Output the index of the element of $K$, if $K$ is present in $A$, else output $-1$.

---

## Input Format

- The first and only line of input contains a single string $S$.

---

## Output Format

- Output the length of the longest substring with no repeating characters.

---

## Constraints

- $2 \leq N \leq 10^5$

---

## Examples

**Example 1**

**Input**

```text
9 7 
1 3 6 7 10 13 20 25 27
```

**Output**

```text
3
```

**Example 2**

**Input**

```text
9 2
1 3 6 7 10 13 20 25 27
```

**Output**

```text
-1
```

**Explanation**

$2$ does not exist in this array
