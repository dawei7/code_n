## Problem Statement

Given a positive integer $N$, the objective is to determine whether integer $N$ is prime or not. If $N$ is prime or $1$, the function should return $1$. Otherwise, if $N$ is not prime, the task is to calculate and return the count of distinct prime factors of $N$.

---

## Input Format

- The first line of input will contain a single integer $T$, denoting the number of test cases.
- Each test case consists of a single line of input.

---

## Output Format

For each test case, output on a new line the number of distinct prime factors of the number or print 1 if the number is 1 or prime.

---

## Constraints

- $1 \leq T \leq 10000$
- $1 \leq N \leq 10000$

---

## Examples

**Example 1**

**Input**

```text
2
121
26
```

**Output**

```text
1
2
```

**Explanation**

- The only Prime factor of 121 is 11 hence the output is 1.
- The Prime Factors of 26 include 2 and 13 hence the output is 2.
