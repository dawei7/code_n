## Problem Statement

Given an integer $N$, output the $N^{th}$ term in the Fibonacci Series.
The Fibonacci Series is : `0 1 1 2 3 5 8...`

### Video Explanation

---

## Input Format

- The only line contains a single integer $N$

---

## Output Format

- Output the $N^{th}$ term in the Fibonacci Series.

---

## Constraints

- $1 \leq N \leq 100$

---
