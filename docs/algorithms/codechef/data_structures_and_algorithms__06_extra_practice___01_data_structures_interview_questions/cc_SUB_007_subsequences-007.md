## Problem Statement

You are given $T$ testcases , in each testcase you are given a binary string $S$ of length $N$ consisting of only $0$'s and $7$'s . Find the number of subsequence that are equal to $``007"$.

A subsequence of a string is a string that is generated by deleting some character of the given string without changing its order.

---

## Input Format

- First line contains $T$ , Number of Testcases

- Next $T$ lines contains a binary string $S$ of length $N$ of each Testcase.

---

## Output Format

- For each testcase print one line consisting of number of subsequences $``007"$

---

## Constraints

- $1 \leq T \leq 100$

- $1 \leq N \leq 10^5$

- Sum of $N$ overall Testcases does not exceed $10^6$

# Counting "007" Subsequences - Detailed Editorial

In this lesson, we explore how to count the number of subsequences equal to `"007"` in a given string. The string contains only the characters `'0'` and `'7'`. We will discuss **two approaches**, explain the ideas behind them, and provide complete code implementations in both C++ and Python.

---

## Approach 1: Direct Counting with Variable Accumulation

### Idea

The key observation is that we can count occurrences of the subsequence `"007"` by iterating through the string and maintaining three counters:
- **$cnt_{0}$**: The number of individual `'0'` characters encountered.
- **$cnt_{00}$**: The number of ways to form the subsequence `"00"` so far.
- **$cnt_{007}$**: The number of ways to form the full subsequence `"007"`.

For each character in the string:
- If the character is `'0'`:
  - Every previous `'0'` can pair with the new `'0'` to contribute to `"00"`. Hence, we add $cnt_{0}$ to $cnt_{00}$.
  - Then we increment $cnt_{0}$ as a new starting point for a subsequence.
- If the character is `'7'`:
  - Each formation of `"00"` so far can be extended with this `'7'` to become `"007"`. Thus, we add $cnt_{00}$ to $cnt_{007}$.

Since we process each character only once, the overall time complexity is $O(N)$ for each test case.

### Code Implementation

#### C++ Code
```cpp
#include
#include
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int T;
    cin >> T;
    while (T--) {
        string S;
        cin >> S;
        long long cnt0 = 0, cnt00 = 0, cnt007 = 0;
        for (char ch : S) {
            if (ch == '0') {
                cnt00 += cnt0;
                cnt0++;
            } else if (ch == '7') {
                cnt007 += cnt00;
            }
        }
        cout << cnt007 << "\n";
    }
    return 0;
}
```

#### Python Code
```python
import sys

def count_subsequences(s: str) -> int:
    cnt0 = 0
    cnt00 = 0
    cnt007 = 0
    for ch in s:
        if ch == '0':
            cnt00 += cnt0
            cnt0 += 1
        elif ch == '7':
            cnt007 += cnt00
    return cnt007

input_data = sys.stdin.read().strip().split()
T = int(input_data[0])
results = []
index = 1
for _ in range(T):
    s = input_data[index]
    index += 1
    results.append(count_subsequences(s))
print("\n".join(map(str, results)))
```

## Approach 2: Dynamic Programming Using a DP Array

### Idea

We use a DP array where each element $dp[i]$ represents the number of ways to form the prefix of the target pattern `"007"` of length $i$. Let the pattern have length $m = 3$.

1. **Initialization:**
   Set $dp[0] = 1$, which represents the empty subsequence.

2. **DP Transition:**
   For every character $c$ in the string, iterate over the pattern in reverse order (from index $m-1$ down to $0$). If $c$ matches the pattern at position $j$, then update:
   $$ dp[j+1] \mathrel{+}= dp[j] $$

3. **Answer:**
   After processing all characters, $dp[3]$ contains the number of subsequences equal to `"007"`.

This method generalizes well to any subsequence counting problem. Since $m$ is constant (i.e., 3), the time complexity per test case is effectively $O(N)$.

### Code Implementation

#### C++ Code
```cpp
#include
#include
#include
using namespace std;

int main() {
    ios_base::sync_with_stdio(false);
    cin.tie(NULL);

    int T;
    cin >> T;
    const string pattern = "007";
    while (T--) {
        string S;
        cin >> S;
        vector dp(4, 0); // pattern length m = 3, so dp size = m + 1 = 4
        dp[0] = 1; // empty sequence
        for (char c : S) {
            for (int j = 2; j >= 0; j--) { // iterate backwards over indices 0,1,2 of the pattern
                if (c == pattern[j]) {
                    dp[j+1] += dp[j];
                }
            }
        }
        cout << dp[3] << "\n";
    }
    return 0;
}
```

#### Python Code
```python
import sys

def count_dp(s: str) -> int:
    pattern = "007"
    m = len(pattern)
    dp = [0] * (m + 1)
    dp[0] = 1  # base case: empty subsequence
    for ch in s:
        for j in range(m - 1, -1, -1):  # iterate backwards through the pattern
            if ch == pattern[j]:
                dp[j+1] += dp[j]
    return dp[m]

input_data = sys.stdin.read().strip().split()
T = int(input_data[0])
results = []
index = 1
for _ in range(T):
    s = input_data[index]
    index += 1
    results.append(count_dp(s))
print("\n".join(map(str, results)))
```

## Conclusion

We have outlined **two approaches**:
- **Approach 1 (Direct Counting):** Uses three counters and runs in $O(N)$ time.
- **Approach 2 (Dynamic Programming):** Uses a DP array to count subsequences for any pattern; also runs in $O(N)$ time.

For practical purposes, both approaches are efficient and clear methods for counting the `"007"` subsequences in a string. This concludes our lesson on counting `"007"` subsequences!
