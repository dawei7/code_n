# Algorithm Docs Layout

Algorithm references are grouped by source family before category:

- `neetcode/`: NeetCode challenge docs, currently organized by NeetCode topic folders such as `neetcode_arrays` and `neetcode_dp1`.
- `geeksforgeeks/`: General algorithm and data-structure docs, organized by category folders such as `graphs`, `dynamic`, and `sorting`.
- `leetcode/`: LeetCode metadata plus original local reference scaffolds. Keep official URLs and stable metadata here, but do not copy LeetCode statements, editorials, or solution text.

When adding new docs, keep the source-family boundary first so future editing sessions can load only the relevant subtree.
