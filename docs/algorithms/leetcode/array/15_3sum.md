## Problem Description & Examples
### Goal
You are given a string `s` and an integer `k`. You can replace at most `k` characters in the string with any uppercase letter. Find the length of the longest substring containing the same letter after performing the replacements.

### Function Contract
**Inputs**

- `s`: str - uppercase letters
- `k`: int

**Return value**

int - length of longest valid window

### Examples
**Example 1**

- Input: `s = "AABABBA", k = 1`
- Output: `4`

**Example 2**

- Input: `s = 'D', k = 1`
- Output: `1`

**Example 3**

- Input: `s = 'AD', k = 0`
- Output: `1`
