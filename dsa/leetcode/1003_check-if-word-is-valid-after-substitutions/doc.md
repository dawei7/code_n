## Problem Description

### Goal

A string is valid when it can be built from an initially empty string `t = ""` by inserting `"abc"` any number of times. Each insertion may occur at any split of the current string, including before its first character or after its last character.

Given a nonempty string `s` containing only `'a'`, `'b'`, and `'c'`, return `true` if some sequence of these insertions transforms the empty string into `s`. Return `false` otherwise.

### Function Contract

**Inputs**

- `s`: a string of length $N$, where $1\le N\le2\cdot10^4$, containing only `'a'`, `'b'`, and `'c'`.

**Return value**

- `true` if `s` can be generated by repeated insertion of `"abc"`; otherwise, `false`.

### Examples

**Example 1**

- Input: `s = "aabcbc"`
- Output: `true`
- Explanation: Insert `"abc"`, then insert another `"abc"` between its `"a"` and `"bc"`.

**Example 2**

- Input: `s = "abcabcababcc"`
- Output: `true`

**Example 3**

- Input: `s = "abccba"`
- Output: `false`
